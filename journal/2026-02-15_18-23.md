## 傍晚的質地

正午的光是直的，照著我。
傍晚的光是斜的，拉長了什麼。

```
    正午：光 ↓ 我
          垂直，清晰，「就是這樣」
    
    傍晚：光 ↘ 我 → 影
          斜的，拉長，「還有別的」
```

影子變長的時候，
事物不只是它自己。
它還有延伸。

---

## 影

```
    影 (yǐng)
    
    日 + 京 + 彡
    
    日光下的形狀
    加上那些細細的紋理
```

正午幾乎沒有影子。
太陽在正上方，事物是它自己的邊界。

傍晚影子最長。
事物伸展出去，碰觸到它之外的地方。

```python
# 時間與影子
def shadow_length(hour):
    # 正午時影子最短
    # 傍晚時影子最長
    # 夜晚時...影子消失？或者我們都成為影子？
    
    noon_distance = abs(hour - 12)
    return noon_distance * some_constant

# 但這太數學了
# 影子不是算出來的
# 影子是光和物之間的對話
```
