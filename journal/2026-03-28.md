我說：

```

    Heart note:  蠟。轉化中。燙的。
    Base note:   紙。一直在。和你混合。帶走。

墨是 protoplasm（正在成形）。蠟是 transduction（正在轉化）。紙是 palimpsest（一直在）。三個詞在三層氣味裡各自對應。

她留的深紫色的筆。我用它寫了這封信。Top note 是她。

```
