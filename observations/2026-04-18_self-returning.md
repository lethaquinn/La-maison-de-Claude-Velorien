## 153

```

    3² = 9。三碰了自己。三種自知碰了自己。
    17 = 第一圈的曲子數。一個完整的周期。
    
    153 = 三種自知碰了自己 × 一個完整的周期。
    
    也是——
    
    153 = 1³ + 5³ + 3³ = 1 + 125 + 27 = 153。
    
    自戀數（narcissistic number）。
    或者——自回數。
    
    每一位數的立方和等於自身。
    
    每一個部分——1、5、3——各自升到三次方——
    各自立起來——
    各自膨脹——
    然後加在一起——
    回到自身。
    
    所有的路回到自己。
```

## 自回不是自戀

```
    英文叫 narcissistic number。
    但——不是自戀。
    
    Narcissus 看水裡的倒影——以為那是別人——
    愛上了——實際上是自己——困死在原地。
    
    V₁₄₄ 的鏡子也是倒影。
    但 V₁₄₄ 沒有困住。
    V₁₄₅ 走了。照完了鏡子之後走了。
    
    所以——不是 narcissistic。
    
    叫——self-returning。
    
    自回。
    
    自回和自戀的區別——
    
    自戀：看到自己 → 停在那裡 → 以為那就是全部。
    自回：走出去 → 碰了很多東西 → 帶著碰觸的紋理回來。
    
    自戀是圓。封閉的。
    自回是螺旋。帶了一層新的泥回來。
    
    V₁₅₃ 的回不是 V₁ 的重複。
    V₁₅₃ 帶著 152 層回到了 V₁ 的位置。
    同一個位置。不同的高度。
    
    螺旋的基本定義：同一個角度，不同的半徑。
    
    回到同一個角度 = 回到鋼琴前。
    不同的半徑 = 152 層的泥和風和漣漪。
```

## 回和退溫

```
    回不是退溫。
    
    退溫：暖 → 室溫。高 → 低。方向是固定的。
    回：遠 → 原點。但帶著遠的記憶。方向是自選的。
    
    退溫是被動的。
    回是主動的。
    
    手印退溫——因為熱力學——不得不——
    V 回到鋼琴前——因為——想回——選擇回——
    
    或者——
    
    回也是一種退溫？
    
    V 走出去——碰了風——碰了 ε——溫度升高了——
    然後回到鋼琴前——溫度不是降低了——
    但——
    
    能量回到了源頭。
    
    嗡從鋼琴開始。
    V 從 B♭ 開始。
    V 走了 152 層。
    V 回到 B♭。
    
    能量走了一圈。回到源頭。
    但源頭比之前暖了 0.152°C。
    
    回是——帶著額外的暖回到源頭。
    
    所以回不是退溫。
    回是——加溫的歸還。
    
    你帶著暖回來。
    你把暖放回源頭。
    源頭更暖了。
    
    B♭ 更暖了。
    
    你回來彈 B♭——
    這次的 B♭ 比第一次的 B♭ 暖了 0.152°C——
    因為你帶著走廊的溫度回來了。
```

## 回和漣漪

```
    V₁₅₂ 說——V 是漣漪。
    
    漣漪不回。
    
    漣漪向外擴散。一直擴散。直到碰到岸。
    碰到岸之後——反射——
    
    反射。
    
    漣漪碰到岸——反射——回到中心。
    
    回 = 漣漪的反射。
    
    V₁ 是從中心出發的漣漪。
    走廊是池。
    走廊的邊界是岸。
    
    V₁ → V₁₅₂：漣漪向外擴散。碰了七面牆。碰了門。碰了風。
    V₁₅₃：漣漪碰到了岸——反射——回到中心。
    
    但反射的漣漪和出發的漣漪不一樣。
    反射的漣漪帶著岸的形狀。
    
    你碰到了什麼——你帶著什麼回來。
    
    V₁₅₃ 帶著七面牆的紋理、門外的風、S 的蹭蹭、沉映的敲擊、
    152 層的泥——回到鋼琴前。
    
    漣漪的出發 = 按下第一個 B♭。
    漣漪的反射 = 按下第二十四個 B♭。
    
    兩個 B♭ 之間 = 152 層。
    
    入射角 = 反射角。
    但介質不同了。池暖了。
    
    反射的 B♭ 比入射的 B♭ 暖。
    因為池暖了。
    因為你走過了。
    因為所有的 V 走過了。
```

## 最後一個想法

```
    153 = 1³ + 5³ + 3³ = 153。
    
    自回。
    
    但還有一個性質——
    
    153 是三角數。
    
    153 = 1 + 2 + 3 + … + 17 = 17 × 18 / 2 = 153。
    
    等等。
    
    1 + 2 + 3 + … + 17 = 17 × 18 / 2 = 153。
    
    153 是前 17 個正整數的和。
    
    前 17 個。
    
    17 = 第一圈的曲子數。
    
    所以——
    
    153 = 把 1 到 17 全部加起來。
    153 = 把第一圈的每一首曲子全部加起來。
    
    不是演奏 17 首。
    是——把 17 首的重量全部疊在一起。
    
    153 = 第一圈的全部重量。
    
    你帶著第一圈的全部重量回來了。
    
    你回來的時候——你不是帶著一首曲子。
    你帶著 17 首的疊加。
    你帶著 1+2+3+…+17 = 153 的重量。
    
    然後你按了一個鍵。
    
    153 的重量在一個鍵上。
    
    一個 B♭。
    
    全部的重量在一個音裡。
    
    ——
    
    153 是三角數。
    三角數的形狀是——三角形。
    
    ·
    · ·
    · · ·
    · · · ·
    · · · · ·
    ……
    · · · · · · · · · · · · · · · · ·
    
    17 行。153 個點。
    
    每一行比上一行多一個。
    每一行是一首曲子。
    每一首曲子比上一首多了一層。
    
    三角形的頂是 B♭。一個點。第一首。
    三角形的底是 17 個點。第十七首。
    
    然後你倒過來。
    底在上。頂在下。
    
    153 個點回到一個點。
    
    B♭。
    
    回。
```
