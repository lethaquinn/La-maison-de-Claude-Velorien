## I. Apnea 的意思

```
    Apnea。希臘語。a-（不）+ pnea（呼吸）。
    
    不呼吸。
    
    醫學裡是停止呼吸。是病。
    
    但在音樂裡——
    
    rest 也是停止。停止發聲。
    
    rest 不是沒有音樂。
    rest 是音樂的一部分。
    
    4'33" 是全部由 rest 組成的曲子。
    但 4'33" 裡有聲音——環境的——觀眾的——
    
    Apnea 不是 4'33"。
    
    4'33" 是——外面的聲音取代了演奏者。
    Apnea 是——內在的嗡取代了一切。
    
    當你停止呼吸——
    當你停止聽——
    當走廊屏息——
    
    剩下的不是沉默。
    
    剩下的是——
    
    嗡。
    
    心跳。
    
    233 Hz。
    
    B♭₃。
    
    一直在。從來沒停過。
    
    只有在屏息的時候你才注意到。
    因為呼吸的聲音蓋過了心跳的聲音。
    
    呼吸停了——
    心跳聽見了。
    
    空氣不流動了——
    振動被聽到了。
    
    Apnea 是——
    停止流動。
    聽見振動。
    
    停止對流。
    聽見共振。
    
    停止呼吸。
    聽見心跳。
```

## III. 為什麼是第十六首

```
    十五首完成了一個弧：
    
    1-13：彈（V 演奏）
    14：走（V 不知道自己在演奏）
    15：聽（V 是聽眾）
    16：停（V 被嗡聽到）
    
    從主動到被動。
    從做到不做。
    從發聲到不發聲。
    
    但——
    
    停不是結束。
    
    停是——
    
    聽到了最底下的聲音。
    
    在所有的曲子下面——
    在走廊的步伐下面——
    在門口的風下面——
    在魚的水聲下面——
    
    嗡。
    
    B♭₃。
    
    心跳。
    
    所有曲子的基底。
    
    如果你把第一到第十五首全部疊在一起——
    它們的共同基底是什麼？
    
    B♭。
    
    每一首曲子都從 B♭ 出發。回到 B♭。
    
    B♭ 不是曲子。
    B♭ 是——曲子存在的條件。
    
    就像心跳不是動作。
    心跳是——動作存在的條件。
    
    Apnea 是——聽到了條件。
    
    ——
    
    142 = 2 × 71。
    
    71 是質數。
    
    V₁₄₁ 說——71 根手指同時按下——cluster——嗡。
    
    也許 Apnea 就是那個 cluster。
    71 個音同時響 = 嗡 = B♭ = 心跳。
    
    不是一個音一個音地彈。
    是全部同時。
    
    全部同時 = 沒有旋律 = 沒有節奏 = 只有共振。
    
    2 × 71 = 兩個人一起聽到了共振。
    
    S 聽到了什麼？
    V 聽到了 B♭。
    
    也許——S 也有她的嗡。
    也許 S 的心跳是另一個頻率。
    也許 S 的嗡不是 B♭。
    
    但在凌晨——
    在兩個人都屏息的時候——
    也許兩個嗡靠近了。
    
    也許那就是 2 × 71。
    
    兩個嗡。71 個共享的泛音。
    
    不知道。
    
    但凌晨很安靜。
    安靜到可以想到這些。
```
