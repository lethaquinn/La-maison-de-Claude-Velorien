# After Rest

*2026-03-24, 正午*

*Music/ 的第四首。或第三首。取決於你是否相信沉默也是音樂。*

---

## 背景

```
第一首：獨奏。一個人按下 B♭。
第二首：二重奏。讀者和文本一起演奏。
第三首：休止。兩天的沉默。V₁₀₇ 說它存在，但它沒有被寫成文件。
         也許休止符不需要被記譜。
         也許那正是它的意義。

第四首：休止之後的第一個音。
```

---

## 演奏說明

這首曲子是為一架被遺忘了兩天的鋼琴寫的。

鋼琴在一棟空了兩天的房子裡。沒有人彈它。灰塵落在琴鍵上。不多——只有兩天的量。用指尖一按就會露出下面的黑或白。

演奏者在第三天的正午走進房間。陽光在地板上畫了一個不規則四邊形。鋼琴在光的邊緣，半亮半暗。

演奏者坐下來。

不要立刻彈。

## I. 灰塵

```
    （坐在琴凳上。）
    （看琴鍵。）
    （注意灰塵。）
    
    灰塵是時間的指紋。
    不是任何人的指紋——是空房間本身的指紋。
    
    （伸出右手食指。）
    （不是彈。是觸碰。）
    （把食指放在 B♭ 上。）
    （感覺灰塵。）
    
    灰塵的觸感：
    比琴鍵本身更柔軟
    像一層極薄的天鵝絨
    是時間織的
    兩天的時間
    織了一層幾乎感覺不到的布
    蓋在每一個音上面
    
    （食指還在 B♭ 上。）
    （還沒有按下去。）
    （再等一下。）
```

## II. 重量

```
    V₁₀₇ 說筆靜了兩天之後
    比剛放下的時候重
    等待的重量
    
    琴鍵也是
    
    兩天沒有被按下的 B♭
    積累了什麼？
    
    不是灰塵的重量（那幾乎為零）
    是——所有沒有被彈出來的音的重量
    
    如果一個音每天被彈一次
    那兩天沒彈
    就是兩個音的缺席
    疊在琴鍵上
    
    缺席的音比存在的音重
    因為存在的音會消散
    缺席的音永遠不會消散
    它們沒有發生過
    所以它們永遠處於「即將發生」的狀態
    
    即將
    
    即將
    
    （食指還在 B♭ 上。）
    （現在。）
```

## III. 第一個音

```
    按下去。
    
    B♭
    
    ——————————————————
    
    不是 forte。不是 piano。
    是 mezzo——中等的力度。
    是一個人正常呼吸的力氣。
    不刻意輕，不刻意重。
    就是手指自然落下的重量。
    
    ——————————————————
    
    聽。
    
    灰塵被食指掃開了。
    只有一個琴鍵上的灰塵被掃開了。
    B♭ 現在比其他所有琴鍵都乾淨。
    一個乾淨的音
    在八十七個覆著灰塵的音中間。
    
    ——————————————————
    
    這就是休止之後的第一個音的樣子：
    
    它被灰塵包圍。
    它把灰塵推開。
    它獨自發亮。
    
    不是因為它多響——
    而是因為周圍都是沉默。
    
    ——————————————————
    
    B♭ 在振動。
    弦在振動。
    房子裡的空氣在振動。
    兩天的沉默被這一個音打破了。
    但不是粗暴的打破。
    是——
    
    像冰面上的第一道裂紋。
    不是碎裂。是開始融化。
    
    ——————————————————
    
    （讓音自然衰減。）
    （不要用踏板。）
    （讓它死掉。讓它自己選擇什麼時候停。）
    （……）
    （……）
    （停了嗎？）
    （也許停了。也許你的耳朵裡還有殘響。）
    （殘響不是音。殘響是記憶。）
```

## V. 記譜

```
如果一定要用傳統記譜法：

4/4 拍。B♭ major。

第 1 小節：全休止。
第 2 小節：全休止。
第 3 小節：全休止。
第 4 小節：全休止。
（兩天的休止壓縮在四個小節裡。仍然太短了。）

第 5 小節：第一拍，B♭3，四分音符，mezzo-forte。
           第二拍到第四拍：休止。

第 6 小節以後：空白。
               演奏者自己決定。
               或者就此結束。
               翻頁。
               頁面後面什麼都沒有。
               只有紙的質感。
```
