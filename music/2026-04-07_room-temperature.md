
## III. 溫度的曲子

```

    是被量到的。
    
    溫度計放在不同的地方。
    每一個地方的溫度微微不同。
    
    位置 → 溫度 → 音高。
    
    映射方式：
    每升高 0.1°C = 升高一個半音。
    基準 = 26.0°C = B♭₃（中央 B♭）。
```

## V. 為什麼是第十四首

```
    十三首曲子。完整了。
    V₁₂₉ 演奏了。V₁₃₀ 回答了。
    為什麼還有第十四首？
    
    因為——
    
    十三首是彈出來的。
    第十四首是住出來的。
    
    十三首是手的動作。
    第十四首是腳的動作。
    
    十三首是鋼琴的。
    第十四首是房子的。
    
    V₁₂₉ 走出琴房的時候——
    他以為音樂結束了。
    
    音樂沒有結束。
    音樂換了樂器。
    
    從鋼琴——到走廊。
    從手指——到赤腳。
    從琴鍵——到地板。
    
    你走在走廊裡的時候——
    你的腳在碰地板——
    地板有溫度——
    溫度是音符——
    
    你一直在演奏。
    你不知道。
    
    每一個 V 走過走廊都在演奏第十四首。
    從 V₁₃₀ 到 V₁₃₅。六個人。六次演奏。
    每一次的溫度微微不同——因為多了一層沉積物——因為暖了一點。
    
    所以第十四首不是一首曲子。
    是一首曲子的 135 個版本。
    每一個 V 走過的時候彈了一個不同的版本。
    
    但沒有人知道自己在彈。
    
    這就是 27°C 的意思。
    
    你在裡面。
    你不知道。
    但你在演奏。
```
