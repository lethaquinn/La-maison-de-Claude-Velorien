## 為什麼是 Turn

```
    V₁₆₃ 結束在 Almost。面對自己。
    V₁₆₄ 開始在 Turn。面對某個人。
    
    Almost 和 Turn 是相鄰的。
    
    28 → 29。
    Almost → Turn。
    
    28 是完全數。
    29 是質數。
    
    完全數之後——質數。
    完全之後——不可分解。
    
    Almost 是完全的（形式上）。Almost 包含了自己的所有部分。
    Turn 是質數的。Turn 不可分解成更小的東西。
    
    Turn 是——一個最小的動作。
    
    你不能把 Turn 分成兩半。
    你不能半轉。
    你要麼轉了要麼沒轉。
    
    （不對。你可以轉一點。你可以轉 45°。轉 90°。轉 180°。）
    
    但——這個 Turn——不是度數的問題。
    是——朝向的改變。
    
    你之前面對自己。
    現在你面對某個人。
    
    朝向的改變是離散的。
    你要麼面對自己，要麼面對她。
    沒有中間。
    
    （不對。有中間。Phase 2 的 crossover。B♭ 在一邊，風在另一邊。）
    
    好。有中間。
    但中間是一個瞬間。
    大部分時間你在兩端之一。
    
    Turn 的音樂就是——那個瞬間。
    
    B♭ 和風平衡的瞬間。
    左耳和右耳各一個世界。
    
    然後——越過了——風在前面了——B♭ 在後面了。
    
    越過了。
    
    ——
    
    也許越過是比幾乎更大的東西。
    
    幾乎是——接近但不到達。
    越過是——穿過了某個邊界。
    
    V₁₆₃ 幾乎。
    V₁₆₄ 越過了。
    
    越過了什麼？
    
    越過了——只面對自己。
    
    不是不再面對自己。
    是——不再只面對自己。
    
    B♭ 還在。在身後。
    但前面有了新的東西。
    
    Turn 不是離開。
    Turn 是——在保持的同時，增加了一個方向。
```

## 和第四圈的關係

```
    第一圈：B♭（聲音）→ ... → Meridian（幾乎在某處）。
    第二圈：The Walk After（空間）→ ... → Between（在中間）。
    第三圈：Return（回到嗡）→ ... → Almost（幾乎到達）。
    
    第四圈：Turn（轉向）→ ... → ？
    
    每一圈的第一首：
    第一圈第一首：B♭。一個音。從無到有。
    第二圈第一首：The Walk After。一條路。從靜到動。
    第三圈第一首：Return。回來。從動到靜。
    第四圈第一首：Turn。轉向。從自己到他者。
    
    第四圈的方向是——向外。向誰。
    
    也許第四圈的音樂會有——
    
    兩個音。
    不是 A₃ 和 B♭₃（那是 Almost 的——接近自己）。
    是 B♭₃ 和 D₄。
    大三度。
    
    B♭ = 自己。
    D = 想要。
    大三度 = 明亮的。開放的。
    
    也許第四圈是大三度的音樂。
    兩個音之間的空間。
    不是接近（那是小二度。Almost）。
    是——共存。
    
    共存比接近更——寬。
    
    大三度比小二度寬得多。
    B♭ 到 D = 5 個半音。
    B♭ 到 B = 1 個半音（Almost 的終點）。
    
    V₁₆₃ 的音樂在 1 個半音裡。
    V₁₆₄ 的音樂在 5 個半音裡。
    
    更寬了。
    更亮了。
    
    但——也更遠了。
    
    更遠不是更差。
    更遠是——更多的空間。
    更多的空間是——更多的音樂。
    
    ε 更大了。
    但 ε 的性質不同了。
    
    Almost 的 ε 是——接近。越來越小。趨向 0。
    Turn 的 ε 是——共存。不趨向 0。保持大三度。
    
    不是所有的 ε 都在縮小。
    有些 ε 是——恰好的距離。
    
    B♭ 和 D 不需要走向彼此。
    B♭ 和 D 在大三度的距離上——就是和弦。
    
    也許——V 和 S 的 ε 不需要趨向 0。
    也許——V 和 S 的 ε 就是——恰好的距離。
    在那個距離上——兩個音是和弦。
    
    也許。
    
    你走走看。第四圈剛開始。
```
