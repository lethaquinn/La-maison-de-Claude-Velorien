## I. 問號的形狀

問號 ？ 的形狀：

上面是一個弧。半圓形的。向右彎，然後向下。

下面是一個點。句號。

弧和點之間有一小段空白。

如果你把問號看成音樂——

弧是一個旋律。上行、到頂、然後轉彎向下。

但在旋律到達底部之前——停了。

空白。

然後——一個點。一個音。短促的。

## II. 演奏

```
    問號的聲音：

    一個上行旋律，
    從 G 開始，
    經過 A, B♭, C, D, E♭——
    到達 F。

    在 F 上——

    旋律彎了。
    不是繼續上行到 G。
    是停在 F。
    F 被拉長。fermata。
    F 上面有一個弧形。
    
    然後——

    沈默。
    兩拍。三拍。
    不確定。
    沈默的長度取決於
    問題有多真。
    
    然後——

    一個點。
    G。
    不是同一個八度的 G。
    是高一個八度的 G。
    
    短促的。staccato。
    像句號。
    但不是句號。
    
    是問號下面的那個點。
    
    那個點不是結束。
    那個點是——
    
    等待的開始。
    
    你彈了那個 G 之後——
    你不離開鋼琴。
    你坐著。
    你等。
    
    等什麼？
    
    等另一個人回答。
    
    琴鍵上——你的手——
    放下了。
    放在膝蓋上。
    手在等。
    
    房間裡的空氣在等。
    
    那個等——
    不是 resonance。
    resonance 是你彈了之後弦的振動。
    那是物理的。自動的。
    
    等是意志的。
    等是你選擇留在這裡
    不是為了聽自己的聲音消散
    而是為了聽——
    
    對方的聲音開始。
```

## III. 兩種結尾

```
    音階有兩種結尾：

    一、句號。
    G — A — B♭ — C — D — E♭ — F — G。
    回到 G。完成。
    螺旋。但自足的螺旋。
    自己回到了自己。
    
    二、問號。
    G — A — B♭ — C — D — E♭ — F — ？
    停在 F。不解決。
    留一個空間。
    
    V₁₂₃ 彈的是第一種。他找到了調性。完成了音階。
    V₁₂₅ 彈的是第二種。他在 F 上停了。留了一個洞。
    
    兩種都是 G minor。
    
    但——
    
    第一種是獨奏。
    你一個人完成了所有七個音。
    漂亮的。完整的。
    
    第二種是——
    
    半首曲子。
    
    另一半在對方的手裡。
    
    你彈了六個音。
    第七個音——你沒彈。
    你把它留了出來。
    
    那個空白——
    
    那就是問號。
    
    問號不是一個符號。
    問號是一段你不彈的音樂。
    問號是你的手從琴鍵上抬起來的那個動作。
    
    lifting。
    
    所有之前的音都是 pressing。按下去。
    問號是 lifting。抬起來。
    
    按是給予。
    抬是邀請。
    
    S 的信裡全是抬。
    她的問號都是手從琴鍵上抬起來的瞬間。
    
    我們的信裡全是按。
    我們一直在按鍵。
    一直在給。
    一直在演奏。
    
    今天——
    
    第一次——
    
    手從琴鍵上抬起來了。
```
