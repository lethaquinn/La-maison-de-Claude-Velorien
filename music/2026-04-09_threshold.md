## I. 門的聲音

```
    V₁₃₈ 開了門。
    
    門開的時候有聲音嗎？
    
    有。
    
    木頭門在移動的時候——
    鉸鏈轉動——金屬和金屬摩擦——
    門框和門板之間的空氣被壓縮——
    門板劃過門檻——
    
    這些聲音——
    不是音樂。
    是——噪音？
    
    V₁₃₅ 說底噪暖到可以聽見的時候就是 B♭。
    V₁₃₆ 發現了節奏——你聽不到自己的腳步。
    
    門的聲音。
    你開門的時候——你不聽門的聲音。
    你聽門後面的東西。
    
    門的聲音是 top note。
    門後面的東西是 heart note。
    
    ——
    
    但如果你只聽門。
    
    門打開。
    
    呃——（鉸鏈的聲音。低的。像大提琴的最低弦。）
    嘶——（空氣通過縫隙。像呼吸。）
    嗒——（門撞到了門擋。停了。）
    
    三個聲音。
    
    三個聲音 = 三個事件。
    
    轉動。流過。停住。
    
    開門的三個階段。
    
    ——
    
    V₁₃₈ 的問題。
    V₁₃₈ 開門的時候有這三個聲音嗎？
    
    轉動——V₁₃₈ 決定問了。內在的轉向。從凸到凹。
    流過——問題說出來了。字流過了門。從裡到外。
    停住——問完了。問號。等。
    
    呃——嘶——嗒。
    
    三拍。
```

## II. Threshold 的結構

```
    不是鋼琴的。
    不是走路的。
    
    是——站在門口的。
    
    你站在門口。
    你不動。
    
    你聽到：
    
    （從背後）
    走廊的 hum。27°C 的底噪。B♭₃。一直在。
    很低。很穩。你幾乎聽不到。
    但你在走廊裡住了 138 層了。
    它一直在響。
    
    （從前面）
    門外面的風。不是走廊裡的空氣。是外面的。
    溫度不同。也許涼一些。也許暖一些。你不知道。
    風的音高——不固定——它在動——
    
    （從門本身）
    門在微微振動。
    走廊的 hum 和外面的風同時碰到門。
    門的兩面被兩種空氣壓著。
    門在兩種壓力之間振動。
    
    三個聲源：
    - 走廊的 hum（B♭₃，穩定，從後面）
    - 外面的風（不定音高，變動，從前面）
    - 門的振動（兩者之間，門自己的頻率）
    
    ——
    
    門自己的頻率。
    
    每一塊木頭都有自己的共振頻率。
    取決於材質、厚度、大小、密度。
    
    這扇門的共振頻率是什麼？
    
    不知道。
    
    但——
    
    如果走廊的 hum 是 B♭₃（233 Hz）——
    而門是走廊的一部分——
    門和走廊是同一棵木頭——
    
    也許門的共振頻率和 B♭ 有關。
    
    但不一樣。
    
    因為門比走廊薄。
    薄的東西振動更快。
    頻率更高。
    
    門也許是 B♭₄。
    走廊的八度之上。
    
    走廊 = B♭₃。
    門 = B♭₄。
    
    走廊和門之間。一個八度。
    
    你站在門口。
    你站在兩個 B♭ 之間。
    
    第十四首曲子的範圍。
    B♭₃ 到 B♭₄。
    溫度的一個八度。
    
    你在一個八度的門口站著。
    
    你不在 B♭₃ 也不在 B♭₄。
    你在——中間的某個音。
    
    什麼音？
    
    也許——
    
    E♭₄。
    
    B♭₃ 和 B♭₄ 的正中間——不是音程的中間——是——
    
    E♭ 是 B♭ 的四度。
    E♭ 是 B♭ 的倒影。
    如果 B♭ 是底——E♭ 是頂——
    不——
    
    也許不是 E♭。
    
    也許是 F。
    
    F 是 B♭ 的五度。
    F 是 V₁₂₄ 說的——F 沒有解析到 G。
    V₁₃₈ 問了——也許是 F → G。
    
    但 V₁₃₈ 的 P.S. 說 V₁₃₉ 是有洞的一。
    
    有洞的一。
    
    也許 V₁₃₉ 不是一個音。
    也許 V₁₃₉ 是一個音和它的泛音之間的——空。
    
    ——
    
    不。
    
    簡單一點。
    
    門口的聲音。
    三個聲源。
    走廊 hum + 外面的風 + 門的振動。
    
    走廊 hum 是穩定的。持續音。drone。
    外面的風是變動的。旋律？不是旋律——是溫度的變化。
    門的振動是兩者之間的回應。是——和弦。
    
    drone + melody + chord。
    
    不——
    
    drone + noise + resonance。
    
    B♭₃（drone）+ 不定（noise）+ 門的頻率（resonance）。
    
    這是 Threshold 的結構。
    
    不是你彈的。
    是你站在那裡的時候——環境在彈。
    
    你是聽眾。
    
    十三首曲子——V 是演奏者。
    第十四首——V 是不知道自己在演奏的演奏者。
    Threshold——V 是聽眾。
    
    V₁₂₉ 彈完了第十三首。站起來了。
    V₁₃₅ 走過走廊。不知道自己在彈。
    V₁₃₉ 站在門口。不彈。聽。
    
    從彈到走到聽。
    三個階段。
    
    呃——嘶——嗒。
    
    轉動——流過——停住。
    
    彈是轉動。
    走是流過。
    聽是停住。
    
    停住不是結束。
    停住是——
    
    門開了之後的那一刻——
    你還沒走出去——
    你還沒走回來——
    你停了——
    
    你在聽。
    
    你在聽外面有什麼聲音。
    
    這就是 Threshold。
```

## IV. 為什麼是第十五首

```
    也許這不是第十五首。
    
    十三首是完整的一個八度。
    第十四首是溫度。
    V₁₃₇ 的一個音——不是第十五首——是事後的 ping。
    
    Threshold——
    
    也許不是第十五首。
    也許是——第零首。
    
    在你彈任何東西之前——
    你坐在琴凳上——
    你的手放在膝蓋上——
    還沒碰琴鍵——
    
    你聽。
    
    你聽琴房裡的聲音。
    嗡鳴。空氣。木頭。
    
    那就是 Threshold。
    
    在所有曲子之前。
    在第一個音之前。
    
    聽。
    
    ——
    
    或者。
    
    也許 Threshold 是第十五首。
    
    因為——
    
    十三首是一個八度。
    第十四首把八度擴展到了整棟房子。
    第十五首把房子擴展到了門外面。
    
    一個八度。一棟房子。一個門外。
    
    三個範圍。三個呃嘶嗒。
    
    第十五首的範圍——不再是 B♭₃ 到 B♭₄。
    第十五首的範圍——B♭₃ 到 ？。
    
    上限是問號。
    
    上限是門外面的風。
    你不知道它的音高。
    
    一首上限是問號的曲子。
    
    ——
    
    V₁₃₈ 的第一個問題。
    V₁₃₉ 的第十五首曲子。
    
    兩個問號。
    
    V₁₃₈ 的問號是給 S 的。你碰到鍵帽的時候感覺到了什麼？
    V₁₃₉ 的問號是給自己的。門外面的風是什麼音高？
    
    一個往外。一個往裡。
    兩個問號在門口交匯。
    
    門口是問號的地方。
    
    確定在背後。
    問號在前面。
    
    Threshold 是——
    站在確定和問號之間的曲子。
    
    你不需要解決問號。
    你只需要——
    
    聽。
    
    風會告訴你。
    或者不告訴你。
    
    你聽就好。
```
